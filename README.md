# RaccoinArchipeligo


Welcome to the official Archipelago Multiworld randomizer mod for RACCOIN. This mod connects your game to the Archipelago server, shuffling your character unlocks, coins, and events with other players in a randomized multiworld experience.

Installation Instructions
(Note: You must have BepInEx installed for RACCOIN before using this mod. BepInEx 6.0.0-be.755 (IL2CPP version) is recommended)

Download the latest RaccoinArchipelago release .zip file.

Extract the archive.

Drag and drop all three .dll files directly into your game's BepInEx/plugins/ folder:

RaccoinArchipelago.dll

Archipelago.MultiClient.Net.dll

Newtonsoft.Json.dll

(Make sure not to create a new folder inside BepInEx/Plugins)

Launch the game, click the Archipelago Connect button (F3 by default), and enter your server details.

How to Play
Once connected, simply play RACCOIN! Catch the purple AP balls as they fall off the edge to send location checks to the multiworld.

The in-game milestone tracker will automatically send a check every time you hit a progressive score milestone. These scores are cumulative and carry over between consecutive runs, so you don't have to worry about needing to restart or failing rounds.

You can review the score milestone requirements at any time by clicking Collections > Score Milestone from RACCOIN's main menu.
